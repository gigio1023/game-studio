# Playtest Rubric

Use this for internal and external playtest reports.

## Observe

- What research question is being tested?
- Did the player understand the immediate goal?
- Did the player perform the intended action?
- Did the player notice feedback?
- Did the player experience the target emotion?
- Did the player recover from failure?
- Did the player hit an accessibility barrier?
- Did camera, input, UI, visual hierarchy, or audio block comprehension?
- Where did confusion become frustration?
- Where did the build promise something it could not deliver?

## Record

- Build identifier
- Scenario
- Player profile
- Session length
- Input device
- Task success or failure
- Key observations
- Inferences separated from observations
- Top three observed issues
- Top three successful moments
- Severity and confidence
- Retest recommendation
- Evidence links
- Recommended gate verdict
