# Playtest Rubric

Use this for internal and external playtest reports.

## Observe

- Did the player understand the immediate goal?
- Did the player perform the intended action?
- Did the player notice feedback?
- Did the player experience the target emotion?
- Where did confusion become frustration?
- Where did the build promise something it could not deliver?

## Record

- Build identifier
- Scenario
- Player profile
- Session length
- Top three observed issues
- Top three successful moments
- Evidence links
- Recommended gate verdict
