# M1 Protocol Proof Example

## Goal

Prove the smallest loop:

```text
Player action -> rule response -> visible consequence -> next choice
```

## Required Evidence

- One valid player action.
- One invalid player action.
- A visible state change.
- A repeatable way to verify the outcome.

## Gates

- `GPH-PROTOCOL-PROOF`
- `GPH-PLAYER-EXPERIENCE`
