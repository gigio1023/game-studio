# Taste Profile

## Creator Intent

Make a small game about feeling curious in an unfamiliar place without needing
fast reactions or combat mastery.

## Reference Moments

| Reference Moment | What The Creator Liked |
| --- | --- |
| A room that changes after a switch | The feeling that the world is listening. |
| A puzzle where failure teaches a rule | Mistakes feel useful instead of punishing. |
| A quiet locked door at the edge of a map | Curiosity without heavy exposition. |

## Desired Emotions

- curious
- cautious
- satisfied after noticing a rule

## Undesired Friction

- twitch timing
- long dialogue before interaction
- solutions that depend on guessing designer intent

## Player Motivation Signals

The creator seems motivated by mastery through observation. This is an
inference from the reference moments, not a confirmed requirement.

## Complexity Tolerance

Low to medium. The first proof should use one verb and one rule change.

## Session Shape

Short session, about 5 to 10 minutes for a first playable proof.

## Interaction Preferences

- movement
- one interact button
- visible state changes

## Hard Noes

- combat
- inventory management
- timed fail states

## Open Questions

- Should the game feel lonely, cozy, or slightly unsafe?
- Should later puzzles add rules or recombine the same rule?
