# Game Thesis Example

`Sample Project` is a compact room-shifting puzzle game where the player changes
one room property at a time, watches the space rearrange, and uses the changed
state to open a final path.

## Player Role

The player is a visitor learning how a small building responds to their actions.

## Core Pressure

Each action changes the space in a way that can help or block later actions.

## Core Verb

Shift one room rule and step through the changed layout.

## First Playable Proof

The first playable proof must show:

- one player action that shifts a room rule
- one visible consequence
- one failure or blocked path
- one recovery choice

## What This Game Must Not Become

- A broad adventure with many unrelated mechanics.
- A puzzle game with solutions that feel arbitrary.
- A narrative game where story replaces interaction.
- A content-heavy mystery that hides the core verb.
