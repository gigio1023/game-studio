# Game Thesis Example

`Sample Project` is a compact puzzle adventure where the player explores a
changing space, learns a small set of reliable rules, and uses those rules to
open a final path.

## Player Role

The player is a visitor trying to understand an unfamiliar system.

## Core Pressure

Each action changes the space in a way that can help or block later actions.

## What This Game Must Not Become

- A broad adventure with many unrelated mechanics.
- A puzzle game with solutions that feel arbitrary.
- A narrative game where story replaces interaction.
