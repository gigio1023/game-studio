# Examples

Examples in this repository must be fictional, reusable, and small enough to
scan.

Do not include private game plans, unreleased project names, personal production
state, proprietary screenshots, or project-specific ledger paths.

Each example should demonstrate the framework shape without becoming a template
for one particular game.

Good examples show at least one of these paths:

- An existing direction carried into a proof-gated milestone.
- Taste-to-game discovery that produces a small concept slate.
- A brownfield project audit that stops before rewriting direction.

Every example should make the entry mode, authority owner, current proof target,
review route, and next gate easy to find.
