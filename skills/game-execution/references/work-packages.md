# Work Packages

Use work packages to make agent execution bounded.

## Package Shape

Each package needs:

- Goal
- Owner role
- Experience intent
- Files in scope
- Files out of scope
- Required proof
- Internal evidence records
- Verification command or manual check
- Handoff path

## Parallel Rule

Parallel packages must have disjoint write scopes. Review can cross boundaries;
editing cannot.
