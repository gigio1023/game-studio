# Teaching And Curriculum Canon For Beginner Design Judgment
This round focuses on game design pedagogy: how a beginner moves from taste,
reference moments, and vague preference into structured design judgment without
losing creative authorship. The repo already cites much of the canon. The
remaining gap is less "what should agents cite?" and more "how should agents
teach the designer to see?"

This page is split into smaller files so each page stays easy to review and edit.
