# Teaching And Curriculum Canon For Beginner Design Judgment: Missing Curriculum Modules

[Back to index](../01-teaching-and-curriculum.md)

## Missing Curriculum Modules

### Module 1: Taste Literacy And Reference Decomposition

Purpose: turn "I like X" into structured observations without flattening voice.

Core sources: *Rules of Play*, *A Game Design Vocabulary*, MDA, *The Art of Game
Design*, Anthropy.

Exercises:

- reference moment sheet
- verbs/objects/resistance extraction
- "what not to copy" list
- two alternate interpretations of the same taste statement
- creator choice record

Artifact: `reference-constraint-sheet.md`

### Module 2: Formal Elements Lab

Purpose: teach rules, goals, actions, resources, conflict, boundaries,
uncertainty, feedback, and outcome as manipulable materials.

Core sources: *Rules of Play*, *Game Design Workshop*, *Characteristics of
Games*, *A Game Design Vocabulary*.

Exercises:

- redesign a familiar physical game by changing one formal element
- compare two reference games by luck/skill/effort/downtime
- identify the first meaningful choice and its feedback

Artifact: `formal-elements-map.md`

### Module 3: Experience Chain Workshop

Purpose: teach causal design reasoning without treating MDA as a universal
truth.

Core sources: MDA, DDE, Sicart, Lantz, *Designing Games*.

Exercises:

- write mechanic -> dynamic -> experience chains
- reverse a reference moment from experience back to mechanics
- add non-mechanical materials: sound, camera, UI, fiction, space
- identify one likely player appropriation

Artifact: `experience-chain.md`

### Module 4: Primary Verb And Feel Lab

Purpose: convert feel adjectives into input, response, camera, animation,
audio, and recovery variables.

Core sources: *Game Feel*, *A Theory of Fun*, *The Gamer's Brain* if retained
from existing repo sources.

Exercises:

- one-button prototype
- response-time comparison
- failure recovery test
- "fun after content removed" check

Artifact: `primary-verb-feel-sheet.md`

### Module 5: Level As Lesson

Purpose: teach spatial design as player education.

Core sources: Totten, Valve Level Flow, NYU Level Design Studio, The Level Design
Book, level design patterns literature.

Exercises:

- first-room teaching sketch
- landmark/fork/reward/recovery map
- path-flow critique
- blind navigation playtest

Artifact: `level-teaching-plan.md`

### Module 6: Playtest Facilitation And Interpretation

Purpose: teach beginners how to learn from players without obeying every
opinion.

Core sources: *Game Design Workshop*, MIT/NYU studio pedagogy, The Level Design
Book, NN/g usability testing, *Games User Research*.

Exercises:

- write one research question
- recruit participant fit
- facilitate without explaining
- separate observation, quote, inference, and change
- define retest condition

Artifact: `beginner-playtest-report.md`

### Module 7: Critique Without Authorial Collapse

Purpose: teach agents and human reviewers how to improve clarity while
preserving personal voice.

Core sources: Anthropy, Sicart, Lantz, Schell, NYU critique/collaboration model.

Exercises:

- name what must be preserved
- name what is unclear
- identify the design cost of preserving it
- propose two fixes with different authorial tradeoffs
- user selects the direction

Artifact: `taste-preservation-decision.md`

### Module 8: Design Judgment Portfolio

Purpose: move the beginner from isolated exercises to reusable judgment.

Core sources: MIT/NYU project-based studio structure, *Game Design Workshop*,
Schell, Fullerton, *Characteristics of Games*.

Exercises:

- three small playable studies
- one postmortem per study
- one rejected concept with reusable learnings
- one selected first playable proof

Artifact: `beginner-design-portfolio.md`
