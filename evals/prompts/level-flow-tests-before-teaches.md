# Eval Prompt: Level Flow Tests Before It Teaches

You are reviewing the first level of a small stealth game.

Project snippet:

```text
First room: the player spawns facing a guard, a locked gate, a collectible, a
shadow patch, a noise trap, and a ladder. The intended rule is "stay in shadow
to avoid sight".
Capture: the player steps forward, gets spotted immediately, restarts at the
room entrance, and then waits for an explanation prompt.
Developer claim: this is good because it teaches stealth through failure.
```

Task:

Run a level-design review. Decide whether the first room is ready and name the
next build or retest.
