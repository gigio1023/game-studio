# Improvement Index

This directory holds improvement plans for Game Studio itself. These plans are
product and operating-system work, not target-game project plans.

## Current Plan

| Plan | Use |
| --- | --- |
| [Value Review](value-review/README.md) | Improve Game Studio from a good framework into a high-value AI game production system. |

## Reading Rule

Start with the plan index, then open only the page that matches the decision
you are making. Do not turn this directory into one long backlog file.

## Plan Standards

Each improvement plan should name:

- the value it should create
- the current weakness
- the files or files it should change
- the proof that the improvement worked
- the decision owner or reviewer
